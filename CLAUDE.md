# Jaguar

## Design Context

Full strategic and visual context lives in [PRODUCT.md](PRODUCT.md) and [DESIGN.md](DESIGN.md) — read both before any design work.

- **Register:** brand (landing page, `landing.html` / `src/landing/`) is the primary default; the desktop editor itself (`src/`, `App.css`) is a separate `product`-register surface.
- **North Star:** "The Lit Workbench" — dark workshop, one warm ember light source. Craft-tool confidence, not fantasy-tavern kitsch.
- **Known issue, not yet fixed:** the landing page currently has generic-AI-page tells (`.eyebrow` / `.section-kicker` uppercase mono labels above every section, soft-shadow "photo-pinned" screenshot framing) — see DESIGN.md's Do's and Don'ts before extending either pattern.
